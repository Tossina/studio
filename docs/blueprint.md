# **App Name**: DagoPoker

## Core Features:

- User Account Management: Secure registration, login, profile management, and account recovery.
- Financial Transactions: Deposits and withdrawals using multiple payment methods including Mobile Money (Mvola, OrangeMoney, Airtelmoney).
- Poker Game Variants: Support for Texas Hold'em, Omaha, Stud, and Draw poker variants.
- Game Formats: Cash games, multi-table tournaments (MTT), Sit & Go tournaments, and fast poker.
- Real-time Gameplay Interface: Intuitive table design with clear display of positions, cards, betting actions, and chat. Option for multi-table play.
- Fraud Prevention: Systems for fraud detection (angle shooting), anti-cheat algorithms, and 24/7 moderation.
- AI Opponent: A tool using a generative AI to play hands of poker when a real opponent is not available. The tool may also reason about which poker variant to play.

## Style Guidelines:

- Primary color: Vivid green (#50C878) to evoke the classic color associated with poker.
- Background color: Dark forest green (#1A4D2E) creates a sophisticated feel.
- Accent color: Red (#FF0000) to highlight important elements such as call-to-action buttons.
- Body and headline font: 'Inter' (sans-serif) for a modern and neutral look that is suitable for both headlines and body text.
- Simple and clear icons to represent game actions and options.
- Clean, intuitive, and responsive design that works well on both desktop and mobile devices.
- Subtle animations to indicate action turns or game events without being distracting.
